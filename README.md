# 自动化截流 · 找链接（抖音 + 小红书）

**一个脚本**：采集 → 打分筛选 → 输出「需人工手动截流的链接清单」。
只负责**找**，不负责回复——回复是人工的事。输出过的链接**永久排除**，每次跑只给你全新的。

业务逻辑纯 Python 标准库，无需 pip 装额外依赖（仅采集器 MediaCrawler 需一次性安装）。

## 用法（只有一个脚本 `jieliu.py`）

```bash
python3 jieliu.py setup                   # ① 一次性：装好采集器（clone + 依赖 + 补丁 + 自检）
python3 jieliu.py                          # ② 采集抖音+小红书 → 输出新链接到 data/截流链接.csv
```

就这两步。日常只需重复第 ②步。常用选项：

```bash
python3 jieliu.py --platform xhs --max 5   # 只采小红书，每平台最多 5 帖
python3 jieliu.py --get-comment false      # 不抓评论（更快更稳，但少了评论区买家）
python3 jieliu.py --max-comments 10        # 每帖最多抓 10 条评论（降请求量=降封控）
python3 jieliu.py --no-crawl               # 跳过采集，用上次抓到的数据重新筛
python3 jieliu.py --reset-seen             # 清空「已输出」记录（之后会重新给出旧链接）
python3 jieliu.py --selftest               # 自检（不联网、不动你的数据）
```

> 采集那一步会自动拉起浏览器让你**扫码登录**——请用**专门的小号**，别用你将来发评论触达客户的账号。
> 本脚本不碰账号密码、不自动登录、不发布。

## 输出长什么样

`data/截流链接.csv`（按优先级从高到低排好），每行 = 一条待你手动去截流的链接 + 一行上下文：

| 列 | 含义 |
|----|------|
| `score` / `priority` | 分数 / 高中低优先级（从上往下做）|
| `platform` | 抖音 / 小红书 |
| `type` | `帖子`（去帖子下评论）或 `评论者`（直接回复评论区那个问价的人，转化更高）|
| `intent` | 命中的买家意图（多少钱 / 怎么寄 / 求推荐 …）|
| `who` | 帖主昵称 / 要回复的评论者昵称 |
| `link` | **要打开去截流的链接** |
| `what` | 帖子标题 / 该评论者问了啥 |

控制台也会打印前 20 条，方便你直接点。

## 它怎么挑（最值钱的部分，在 `config.json` 调）

`分数 = 关键词分级 + 意图加分(强>弱，评论区×倍率) + 时效 + 评论活跃 + 海外属地`；命中下列直接丢弃：

- **排除**：招聘 / 加盟 / 培训 / 骗局 …
- **噪声丢弃**：货代知识/科普/FBA/B2B/国内大件，以及**作者本身是货代/集运卖家**
- **评论者线索**：评论区「问价/问怎么寄」的人会被单独抽出来（`type=评论者`），可直接回复 TA
- **海外属地**：发帖/评论者在 美/澳/加/英… → 加分（「海运回国真买家」最廉价的判别特征）
- **永久去重**：任一链接被输出过，以后不再出现（记录在 `data/seen.json`）

想把「对口买家」比例拉高，主要调 `keywords.txt`（搜索词，建议用消费者大白话：海运回国 / 留学回国行李 / 美国寄回国 …）和 `config.json`（分级与权重）。

## 文件总览

| 文件 | 作用 |
|------|------|
| `jieliu.py` | **唯一要运行的脚本**：采集 + 打分筛选 + 输出链接 + 永久去重 + 自检 |
| `config.json` | 关键词分级、意图词、排除/卖家/科普词、海外属地、打分权重（**最常调**）|
| `keywords.txt` | 采集用的搜索词 |
| `collectors/setup_mediacrawler.sh` | 一键装好可跑的采集器（被 `jieliu.py setup` 调用）|
| `collectors/patches/` | 让 MediaCrawler 真正跑通的补丁（CDP 自启 / xhshow 签名兼容）|
| `data/截流链接.csv` | **产物**：待人工截流的链接清单（不入库）|
| `data/seen.json` | 已输出过的链接记录，用于永久去重（不入库）|

## 边界与合规

- 只做**发现**：找链接、打分、去重。**不做**回复、自动发送、多账号群控、绕风控。
- 只采**公开**数据、控制频率、用专门小号；评论原文落盘前已**脱敏**手机号/微信/QQ/邮箱。
- 不存他人手机号/微信号等敏感联系方式；数据仅本地自用、不转卖。回复务必每条不同、针对该帖/该人。
